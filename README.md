# 1. 프로젝트 개요

### 📋 서비스 개요
- 커뮤니티 기반의 예적금 상품 조회 및 추천 애플리케이션
- 돼지 저금통과 금융 상품을 연동하여, **재미있고 효과적으로 금융 목표를 달성**할 수 있도록 돕는 서비스입니다.
- **프로젝트 기간:** 2024/11/01 ~ 2024/11/26 (26일간)

### 💰 **서비스 특징**
1. **저축 목표 달성 지원**
    - 돼지 저금통을 통해 사용자가 명확한 금융 목표를 설정하고, 목표를 향한 진행 상황을 시각적으로 확인할 수 있습니다.
2. **맞춤형 금융 상품 추천**
    - 사용자 정보 및 저축 목표를 바탕으로 **개인화된 예적금 상품**을 추천합니다.
3. **커뮤니티 기반**
    - 금융 정보를 공유하고 응원할 수 있는 커뮤니티 기능으로, **재미있고 동기부여를 하는 저축 활동**을 지원합니다.
4. **디자인**
    - 눈이 즐거운 페이지 구성 및 디자인

### 🐽 주요 기능
- **저금통 관리:** 돼지 저금통 생성, 삭제
- **금융 상품 추천 및 탐색:** 개인 맞춤형 금융 상품 추천, 예적금 상품 목록 탐색
- **회원 기능:** 회원가입, 로그인, 회원 정보 수정, 마이페이지, 팔로잉, 팔로잉 및 팔로워 목록 확인, 내가 가입한 예적금 상품 연동 및 금리 차트
- **커뮤니티 게시판:** 게시글 및 댓글 CRUD, 좋아요 및 저장 기능
- **응원하기:** 새로 생성된 저금통 이용자 응원하기
- **챗봇 서비스:** 금융 지식 설명, 상품 추천 제공
- **주변 은행 검색:** 은행 위치와 정보를 지도와 함께 확인
- **환율 계산기:** 실시간 환율 계산 및 변동 차트 제공

# 3. 기능 상세 설명

## ✅ 금융 상품 추천 알고리즘

로그인한 사용자의 정보와 유사한 가입자들이 선호하는 금융 상품을 추천하는 알고리즘으로, **5000명의 유저 더미 데이터**를 기반으로 작동합니다.

1. **로그인한 사용자의 경우**
    - 유사도 판단 기준: 나이(±5세), 저축 목표, 저축 기간, 저축 금액
      유사한 사용자들이 많이 가입한 예적금 상품을 **금리 순**으로 추천합니다.
    - 추가 필터링 기준:
        - **예금**: 자산 ±2,000만 원
        - **적금**: 저축 금액 ±50만 원
2. **로그인하지 않은 사용자의 경우**
    - 전체 사용자가 많이 가입한 상품 중 **우대 금리** 기준 상위 5개 상품을 추천합니다.

## ✅ 서비스 대표 기능

### 1️⃣ 메인 페이지
![메인 - 서비스 소개](/images/메인%20-%20서비스%20소개.gif)
![메인 - 금융상품추천](/images/메인%20-%20금융상품추천.gif)
![회원 - 커뮤니티 소개, 응원하기](/images/메인%20-%20커뮤니티%20소개,%20응원하기.gif)
- **비회원**
    - **돼지 저금통 기능 소개**
        - 목표 금액 대비 모은 금액에 따라 돼지 저금통의 색상이 변경됩니다.
        - 저축 금액을 무게로 변환해 시각적으로 표시합니다.
    - 전체 사용자가 많이 가입한 **예적금 상품 Top 5**를 제공합니다(예금/적금 전환 가능).
    - 커뮤니티 게시판 소개
- **회원**
    - **개인 돼지 저금통 조회**
        - **저금통이 있는 경우**: 모은 금액, 저금통 이름, 만기일, 상품 정보, 저축 기간, 목표 금액, 응원 받은 수를 확인할 수 있습니다.
        - **저금통이 없는 경우**: “저금통이 없습니다” 메시지와 저금통 생성 버튼을 제공합니다.
    - 회원 정보 기반 **맞춤형 금융 상품 추천 Top 5**를 제공합니다(예금/적금 전환 가능).
- **공통**
    - 돼지 저금통을 새로 생성한 사용자를 응원할 수 있는 기능을 제공합니다.
    - 금융 지식을 설명하고 금융 상품을 추천하는 **챗봇** 기능이 모든 페이지에서 사용 가능합니다.

### 2️⃣ 로그인 및 회원가입 페이지
- **로그인**
    ![회원 - 로그인](/images/회원%20-%20로그인.gif)

    ![회원 - 회원가입](/images/회원%20-%20회원가입.gif)

    - 필수 정보가 누락된 경우, 에러 메시지를 통해 수정하도록 안내합니다.

### 4️⃣ 금융 상품 추천
![금융상품추천](/images/금융상품추천.gif)
- 사용자 정보 기반 금융 상품 추천 리스트를 제공합니다.
- 예적금, 금융사, 저축 기간, 금리로 필터링 가능한 상품 목록을 제공합니다.
- 금융 상품 상세 정보:
    - 상품명, 우대 조건, 은행명, 금리, 상품 설명 등을 포함하며, 공식 홈페이지로 이동할 수 있는 링크를 제공합니다.

![환율-계산기](/images/환율-계산기.gif)

- 범위 내 은행의 이름, 주소, 전화번호, 영업시간 등의 정보를 제공하며, 우측 리스트에서 특정 은행을 클릭하면 지도가 확대되어 정확한 위치를 확인할 수 있습니다.
